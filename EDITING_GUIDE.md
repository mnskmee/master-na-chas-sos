# Руководство по редактированию сайта

## Как редактировать контент

### 1. Изменение телефона
Телефон встречается в нескольких местах:
- `src/components/Header.tsx` - в шапке сайта
- `src/components/Footer.tsx` - в футере
- `src/pages/Contacts.tsx` - на странице контактов
- `src/pages/CallMaster.tsx` - на странице вызова мастера

Найдите `+7 (495) 123-45-67` и замените на ваш номер.

### 2. Изменение адреса и контактов
Отредактируйте файлы:
- `src/components/Footer.tsx` - адрес в футере
- `src/pages/Contacts.tsx` - все контактные данные

### 3. Изменение цветов
Основные цвета задаются в `src/index.css`:
- `--primary` - синий цвет (кнопки, акценты)
- `--secondary` - оранжевый цвет (призывы к действию)

Все цвета в формате HSL. Например:
```css
--primary: 215 85% 45%;  /* синий */
--secondary: 28 90% 55%; /* оранжевый */
```

### 4. Изменение услуг
Список услуг находится в:
- `src/pages/Services.tsx` - подробное описание
- `src/pages/Index.tsx` - краткий список на главной

### 5. Изменение цен
Прайс-лист редактируется в `src/pages/Pricing.tsx`

### 6. Портфолио работ
Примеры работ в `src/pages/Portfolio.tsx`. Изображения находятся в папке `src/assets/`

### 7. Отзывы
Отзывы клиентов редактируются в `src/pages/Reviews.tsx`

## Telegram интеграция

### Где хранятся токены
Токен бота и chat_id НЕ хранятся в коде для безопасности. Они настраиваются через Lovable Cloud.

### Как обновить токены
1. Откройте проект в Lovable
2. Перейдите в раздел Cloud → Secrets
3. Найдите:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота
   - `TELEGRAM_CHAT_ID` - ID чата для получения заявок
4. Нажмите "Edit" и обновите значения

### Как получить токен бота
1. Найдите в Telegram бота @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

### Как получить Chat ID
1. Найдите в Telegram бота @userinfobot
2. Отправьте ему любое сообщение
3. Он ответит с вашим ID (или ID группы, если добавите бота в группу)

### Логика работы формы
Код формы: `src/pages/CallMaster.tsx`
Серверная функция: `supabase/functions/send-telegram/index.ts`

Форма отправляет данные на серверный endpoint, который формирует красивое сообщение и отправляет в Telegram.

## Частые вопросы

### Как изменить логотип?
В файле `src/components/Header.tsx` найдите блок с логотипом (буква "М"). Замените на своё изображение.

### Как добавить новую страницу?
1. Создайте файл в папке `src/pages/`
2. Добавьте роут в `src/App.tsx`
3. Добавьте ссылку в навигацию (`src/components/Header.tsx`)

### Как изменить время работы?
Отредактируйте:
- `src/components/Footer.tsx`
- `src/pages/Contacts.tsx`

### Как изменить текст на главной странице?
Откройте `src/pages/Index.tsx` и отредактируйте заголовки и описания.

## Рекомендации

1. **Всегда тестируйте** изменения перед публикацией
2. **Делайте резервные копии** перед большими изменениями
3. **Проверяйте на мобильных** - сайт оптимизирован под телефоны
4. **Обновляйте контент** регулярно (отзывы, портфолио)

## Безопасность

⚠️ **ВАЖНО**: Никогда не добавляйте токен бота и chat_id прямо в код! Всегда используйте систему секретов Lovable Cloud.
